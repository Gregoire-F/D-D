Le Projet

Le projet Donjons et Dragons est une réalisation de groupe de la classe DWWM 25-26. Les technologies utilisées sont le HTML, CSS, JavaScript et Git pour le déploiement collaboratif via GitHub.

Nous exploitons l'API Dungeons & Dragons : https://www.dnd5eapi.co/api/2014. Ce projet permet aux joueurs et maîtres du jeu de consulter les monstres, sorts, objets, classes, etc., avec des outils utiles tels que des lanceurs de dés et une table de mixage d'ambiance sonore (soundboard).

Url du site : https://gregoire-f.github.io/D-D/

Structure du Projet

DungeonsDragons/

└── README.md         # Documentation technique originale

Installation & Utilisation

  1. Cloner le dépôt : 
    git clone https://github.com/Gregoire-F/D-D.git
  2. Lancer le projet :
    • Ouvrez src/index.html directement dans votre navigateur.
    • Recommandation : Utilisez une extension comme Live Server (VS Code) pour éviter les problèmes de CORS lors des appels API.
  3. Serveur local alternatif : 
    # Avec Python
python -m http.server 8000 --directory src

# Avec Node.js
npx serve src

Fonctionnalités Clés

  • Système de Recherche : Recherche globale via la barre de navigation ou par catégories spécifiques sur la page d'accueil.
  • Soundboard d'Ambiance : Pistes audio variées (Combat, Ambiance, Autre) et mixage multi-pistes pour une immersion totale.
  • Lanceur de Dés : Outil interactif permettant de configurer le nombre de faces et de dés pour des jets instantanés.

Pages Principales

Accueil

Tableau de bord global. Accès direct aux catégories (Héros, Sorts, Items, Monstres, etc.) via des cartes cliquables. La NavBar permet une recherche rapide n'importe où sur le site.

Table de Mixage (Sons)

  • Mode Lecteur : Joue une piste à la fois.
  • Mode Mixage : Superposez plusieurs sons pour créer votre propre atmosphère.
  • Scénarios : Lancez des combinaisons de sons prédéfinies en un clic.

Technique

Utilisation de Web Components personnalisés avec Shadow DOM pour une encapsulation parfaite (ex: DndDiceRoller). Chaque composant gère son propre état et ses animations de manière autonome.
